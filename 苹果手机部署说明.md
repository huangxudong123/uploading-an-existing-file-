# 手机部署到 Cloudflare（不用粘贴代码）

项目只需要上传 3 个文件：

- worker.js
- wrangler.jsonc
- package.json

## 第一步：上传到 GitHub

1. 用 Safari 打开 github.com 并登录。
2. 点击右上角“+” → New repository。
3. 仓库名填写 `agentmail-random-inbox`，建议选择 Private。
4. 创建仓库后，点击 Add file → Upload files。
5. 点击 choose your files，从“文件”App 中选择上面的 3 个文件。
6. 点击 Commit changes。

如果手机网页按钮显示不全：Safari 地址栏左侧点“大小”或“aA” → 请求桌面网站。

## 第二步：在 Cloudflare 导入

1. 登录 Cloudflare。
2. 进入 Workers 和 Pages → 创建应用程序。
3. 选择 Import a repository / 导入存储库。
4. 连接 GitHub，选择刚创建的仓库。
5. 部署命令填写：`npx wrangler deploy`。
6. 其他项目路径保持默认，然后点击部署。

## 第三步：添加密钥

部署后进入该 Worker：

设置 → 变量和机密 → 添加

添加 3 个“机密/Secret”：

- `AGENTMAIL_API_KEY`：新的 AgentMail API 密钥
- `APP_USERNAME`：网页登录用户名
- `APP_PASSWORD`：网页登录密码

保存并重新部署。

注意：不要把 API 密钥写入 GitHub 文件中。
