# 苹果手机更新说明

这个版本已经按你的要求分成两个页面：

- **管理页面**：`https://你的域名/`
  - 创建随机邮箱
  - 查看全部邮箱
  - 复制每个邮箱的独立查看链接
  - 删除邮箱
- **独立邮件页面**：创建邮箱后生成的 `/mail/...` 链接
  - 只显示该邮箱的邮件
  - 没有“创建邮箱”按钮
  - 看不到其他邮箱
  - 不需要进入管理页面

## 更新方法

1. 在 iPhone“文件”App 中解压本压缩包。
2. 用 Safari 打开原来的 GitHub 仓库。
3. 点击“添加文件 → 上传文件”。
4. 上传并替换以下三个文件：
   - `worker.js`
   - `wrangler.jsonc`
   - `package.json`
5. 点击“提交更改”。
6. 回到 Cloudflare 的“部署”页面，等待自动部署显示绿色成功。
7. 刷新你的管理页面。

## Cloudflare 变量

继续保留：

- `MOEMAIL_API_KEY`：MoeMail API 密钥
- `APP_USERNAME`：管理页面用户名
- `APP_PASSWORD`：管理页面密码

旧变量名 `AGENTMAIL_API_KEY` 仍然兼容，不必立刻改名。

可选添加一个“密钥”变量：

- `PUBLIC_LINK_SECRET`：用于生成独立查看链接。建议填写一串较长的随机字符。

不添加也能使用，程序会暂时用 `MOEMAIL_API_KEY` 给链接签名。以后更换 API 密钥时，旧的独立链接可能失效；单独设置 `PUBLIC_LINK_SECRET` 可以避免这种情况。

不需要 KV、D1 或 R2。

## 删除邮箱提醒

管理页面点击“删除”后会要求确认。删除后邮箱、邮件和独立查看链接可能无法恢复。
