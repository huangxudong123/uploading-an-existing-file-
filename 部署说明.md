# 苹果手机更新说明

本版本改动：

- 管理页默认只显示邮箱列表，不再直接显示邮件区域。
- 手动点击邮箱卡片后，才会在当前页面展开邮件列表。
- 再点击具体邮件，会在同一页面显示完整正文，不会跳转。
- 独立查看邮件页面保持不变。
- 页面标题改为“临时邮箱”，移除了顶部“MOEMAIL 管理页面”。
- 管理页面仍可删除邮箱。
- 管理地址可使用 `APP_USERNAME` 和 `APP_PASSWORD` 设置账号密码；独立邮件链接不受影响。

## 更新

1. 解压压缩包。
2. 在原 GitHub 仓库上传并替换：`worker.js`、`wrangler.jsonc`、`package.json`。
3. 点击“提交更改”。
4. 等待 Cloudflare 自动部署成功。
5. 用 `https://你的域名/?v=3` 打开，避免 Safari 使用旧缓存。
