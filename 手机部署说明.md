# 苹果手机更新现有 Cloudflare 邮箱项目

这个版本已经改为调用你的 **MoeMail** 邮箱系统：

- API 地址：`https://mat.work4.ggff.net`
- 请求头：`X-API-Key`
- 创建邮箱：`POST /api/emails/generate`
- 获取邮箱：`GET /api/emails`
- 获取邮件：`GET /api/emails/{邮箱编号}`

## 更新方法

1. 在 iPhone“文件”App 中解压本压缩包。
2. 用 Safari 打开原来的 GitHub 仓库。
3. 进入“添加文件 → 上传文件”。
4. 上传并替换：
   - `worker.js`
   - `wrangler.jsonc`
   - `package.json`（内容没变，上传也没问题）
5. 点击“提交更改”。
6. Cloudflare 会自动重新部署，等待部署显示绿色成功。

## Cloudflare 变量

原来已经添加的三个密钥可以继续使用：

- `AGENTMAIL_API_KEY`：填写 MoeMail 后台生成的新 API Key
- `APP_USERNAME`：网页登录用户名
- `APP_PASSWORD`：网页登录密码

程序也支持把 API Key 变量改名为 `MOEMAIL_API_KEY`，但不是必须，二选一即可。

可选文本变量（通常不用添加）：

- `MOEMAIL_BASE_URL`：默认已经是 `https://mat.work4.ggff.net`
- `MOEMAIL_DOMAIN`：不填时自动读取系统的第一个邮箱域名
- `MOEMAIL_EXPIRY_TIME`：默认 `0`，代表永久；可填 `3600000`（1小时）、`86400000`（1天）、`604800000`（7天）

不需要添加 KV、D1、R2。
